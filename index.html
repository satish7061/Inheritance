Inheritance and Prototype Questions

Qu 1. Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype?

    const parent = {
        firstName: "Satish",
        lastName: "Kumar",
        fullName() {
          return this.firstName + " " + this.lastName;
        }
    };
    const child ={
          __proto__: parent,
    };
    console.log(child.fullName()); 


        Output :

        Satish Kumar

Qu 2. Write code to explain prototype chaining?

    function Person() {
        this.name = 'Satish'
    }
    Person.prototype.name = 'singh';
    Person.prototype.age = 23
    
    const newPerson = new Person();
    
    console.log(newPerson.name); 
    console.log(newPerson.age);


        Output :

        Satish
        23

Qu 4. Write a JavaScript function to retrieve all the names of object's own and inherited properties?

    const parent = {
        age:23,
        rollno: 25,
    };
    const child ={
        __proto__: parent,
        firstName: "Satish",
        lastName: "Kumar",
    };
    for(let prop in child)
        console.log(prop);
